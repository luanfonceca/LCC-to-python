Programação Orientada a Objetos 

** Vamos começar criando um arquivos chamado "classes.py".
Dentro dele, escrava este código:
# coding: utf-8

class Retangulo(object):
    """
    Retangulo(base, altura, cor)
    Classe criada para demonstrar o uso 
    de orientação a objetos com Python!
    """
    
    def __init__(self, base=0, altura=0, cor='Sem cor'):
        self.base = base
        self.altura = altura
        self.cor = cor
        print "Criando nova instância Quadrado"

** Parabéns você acabou de criar sua primeira Classe em Python, 
pra variar, algo muito complicado né?!

** Agora abra o terminal do python no mesmo diretório que
você criou seu arquivo de classes.
Digite então:

>>> import classes
>>> r = classes.Retangulo(base=2, altura=3)
Criando nova instância de Retangulo
>>> r
<classes.Retangulo object at 0x10d80ce90>
>>> r.cor
'Sem cor'
>>> r.cor = 'Azul'
>>> r.cor
'Azul'

** Reveja tudo que fizemos...
- Criamos uma Classe chamada Retangulo;
- Criamos uma instância dela;
- Utilizando o construtor padrão para setar a base e a altura;
- Setando depois uma cor, pois nossa instância não tinha.

** Vamos então adicionar alguns métodos :)
Adicione o método area(), logo após do método __init__():
def area(self):
    return self.base * self.altura

** Para testar, faça:
>>> reload(classes)
<module 'classes' from 'classes.py'>
>>> r = classes.Retangulo(base=2, altura=3, cor='Azul')
Criando nova instância de Retangulo
>>> r
<classes.Retangulo object at 0x10d80ce90>
>>> r.area()
