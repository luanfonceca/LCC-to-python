** Um simples "for" em uma lista.
# range() é uma função geradora, ela gera números, bem rápido :)
>>> lista = range(5)
>>> print lista
[0, 1, 2, 3, 4]

>>> for item in lista:
...     item
...     
0
1
2
3
4
