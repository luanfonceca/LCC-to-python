Condições!

Estruturas Condicionais 

** Um simples "if".
>>> if True:
...     print "Verdade"
...     
É verdade

** E se num for?
>>> if not True:
...     print "Mentira"
... 
>>> # Nada Acontece...

** Usando variáveis
>>> historia = True
>>> if historia:
...     print "Verdade"
... 
Verdade

** Usando o None e negando
>>> historia = None
>>> if historia:
...     print "Verdade"
... elif not historia:
...     print "Mentira"
...     
Mentira

** O "not" é um Modificador, usado em expressões condicionais,
ele permite "afirmar negações".

O que em outras linguagens seria:
if(!historia){
echo("Mentira")
}

** Em Python se torna, muito mais legível...
>>> if not historia:
...     print "Mentira"

** Existem outros, vamos combinar condições com listas!
>>> qualquer = [1, 2, 3, 4, 5]
>>> num = 4
>>> if num in qualquer:
...     print "Tem o 4 ae"
... 
Tem 4 ae

** Usando a função "all()" para turbinar!
boato = [True, True, False]
>>> if all(boato): # Neste caso, tudo precisa ser verdade.
...     print "Tudo Verdade"
... else:
...     print u"Alguém Mentiu"
...     
Alguem Mentiu

** Havendo alguma condição verdadeira o "any()" returna True de tudo
if any(boato):
...     print u"Um já serve"
...
Um ja serve

** Comparando valores
>>> num = 10
>>> if num is 10:
...     print u"é 10 sim!"
...     
É 10 sim!


>>> if num > 5 and num < 15:
...     print u"%s está entre 5 e 15" % num
...     
10 está entre 5 e 15
