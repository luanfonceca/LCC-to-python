Listas 

** Famosa Lista, não vetor, a lista é muuuuuuuito(Over 9000) mais poderosa que um Array bobão.
>>> lista = [1, 2, 3, 4, 5]
>>> print lista
[1, 2, 3, 4, 5]

** Não precisamos criar TAD's para manipular isso, somos jovens!
>>> lista[0]
1
** Viu, é possível indexar, utilizando o Slice.

>>> lista[1:4]
[2, 3, 4]
** Com direito a intervalo e tudo, é o fraco.

** E pode bulír, pode?
>>> lista.append(3) # Adiciona ao final
>>> print lista
[1, 2, 3, 4, 5, 3]

>>> lista.pop() # Remove o último
3

** Bora mais longe?!
>>> lista.append([4, 5, 75])
>>> print lista
[1, 2, 3, 4, 5, [4, 5, 75]]

** Passava o resto do dia, num passava?
>>> lista.insert(0, 'texto')
>>> print lista
['texto', 1, 2, 3, 4, 5, [4, 5, 75]]

** Ei, use o "for" ai homi.
>>> for item in lista:
        print item

Tuplas 

** Não sabe o que é uma Tupla?
A tupla é uma seqüência, como a lista,
porém ela é imutável, ou seja, criou já elvis.
É caracterizada pelos parênteses "()".

>>> tupla = (1, 3, 5, 7)
>>> dir(tupla)
['__add__', '__class__', '__contains__', '__delattr__', '__doc__', '__eq__', '__format__', '__ge__', '__getattribute__', '__getitem__', '__getnewargs__', '__getslice__', '__gt__', '__hash__', '__i
nit__', '__iter__', '__le__', '__len__', '__lt__', '__mul__', '__ne__', '__new__', '__reduce__', '__reduce_ex__', '__repr__', '__rmul__', '__setattr__', '__sizeof__', '__str__', '__subclasshook__'
, 'count', 'index']
>>> print tupla[0]
1
** Ela tem menos métodos né?! Mas saiba que é tão útil quanto a lista :)

>>> tupla2 = (1, 3, 4, 5, 1)
>>> tupla2.count(1)
2 # Duas ocorrências do elemento 1, na tupla2
** Num disse que era útil?!

Grande Dicionário! 

** Você conhece o dict? não?!
O Dicionário, é um dos tipos de dados mais legais existente no python, 
e reza uma lenda que é uma das estruturas de dados mais rápidas também.
Ele é caracterizado por ter uma chave e um valor
algo como {'chave': valor}, onde a chave pode ser uma str ou um int ou um float...

>>> {1: 'a'}
{1: 'a'}
>>> {[1, 2, 3]: 'a'}
Traceback (most recent call last):
  File "<input>", line 1, in <module> 
TypeError: unhashable type: 'list' # Nem tudo são flores ;/
>>> {2.3: 'a'}
{2.3: 'a'}

dicionario = {'a': 1, 'e': 2, 'u': 5}

>>> print dicionario
