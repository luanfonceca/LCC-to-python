Tipos Básicos

Se lembre, os ">>>" representam o shell, tem que digitar lá brother. 
>>> palavra = 'Fulano'
>>> type(palavra)
<type 'str'>	

** Conhece o Slice?!
>>> palavra[:3] # as 3 primeiras letras
'Ful'
>>> palavra[3:] # as 3 últimas letras
'ano'
>>> palavra[::-1] # "Fulano" ao contrário
'onaluF'

** Use um for!

Agora vamos criar arquivos de códigos em python :)
crie um arquivo chamado: "digitar_algo.py"
Encreva isso dentro dele:
# coding: utf-8
texto = raw_input(u"Digite algo: ")
print u"Você digitou: %s" % texto

No terminal, vá até o diretório onde você criou seu arquivo e digite:
$ python digitar_algo.py
